## TypeScript是什么，有什么优缺点
`Typescript`是`Javascript`超集，在JavaScript的基础上添加静态类型定义，支持ES6语法。
### 优点
- 为JS引入了静态类型
- 编译时进行静态检查，能及时发现潜在的错误
- 支持ES6
- 使代码更易维护
### 缺点
- 引用第三方库有可能缺少类型定义，带来不便
- 增加了部分编译时间
